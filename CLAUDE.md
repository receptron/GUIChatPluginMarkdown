# CLAUDE.md

## Plugin Overview

Markdown document display plugin for GUI Chat applications. Render and display Markdown content.

## Common Guidelines

For standard plugin development guidelines, see:
https://github.com/receptron/GUIChatPluginTemplate/blob/main/CLAUDE.md

## Plugin-Specific Notes

### Features
- Markdown rendering
- Syntax highlighting for code blocks
- Support for tables, lists, and other Markdown elements

### Dependencies
- `gui-chat-protocol`: Core protocol for GUI Chat plugins
